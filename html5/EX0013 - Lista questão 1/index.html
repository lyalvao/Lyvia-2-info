O que significa semântica? Por que isso é tão importante em HTML5?

A semântica é o estudo do significado e interpretação de palavras, e aplicando isso ao HTML, significa a melhor compreensão do HTML para o computador. A semântica é importante em HTML5 porque se trata da utilização correta das tags HTML para representar o conteúdo de uma página web de forma clara e significativa.
